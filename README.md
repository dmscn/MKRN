## Installation
```
npm install
```

to install the dependecies.

## Usage

```
npm build
```
to build the dist folder with the code.

```
npm start
```
to start the project.

```
npm dev
```
to run live in dev mode.

## Testing
```
npm test
```
to run all the tests.

```
npm test:live
```
to run tests live.
